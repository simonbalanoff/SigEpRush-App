# ΣΦΕ iOS App

A native iOS application built in Swift to manage Sigma Phi Epsilon rush workflows, including potential new member tracking, event visibility, and internal evaluations.  
The app was designed to support real-time data updates, structured user flows, and consistent state across views.

This app integrates with a [custom TypeScript/Express API](https://github.com/simonbalanoff/SigEpRush-API) to handle data persistence, image uploads, and business logic, keeping the iOS client lightweight and focused on user experience.  
This repository is **public for portfolio and code demonstration purposes only** and is not intended for production or external use.

### Tech
- Swift (Xcode)
- iOS-native UI and state management
- REST API integration
